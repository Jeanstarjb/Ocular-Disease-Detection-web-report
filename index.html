The Problem

The Scalability Crisis in Ophthalmology

Diseases like Diabetic Retinopathy and Glaucoma are leading causes of preventable blindness. Diagnosis relies on manual screening by specialist ophthalmologists, a process that is slow, costly, and inaccessible to remote populations, creating a critical diagnostic bottleneck.

The Data Challenge

Analyzing retinal fundus images is a highly specialized skill. Our challenge was to build an automated system that could analyze these complex images with clinical-grade accuracy, reducing the bottleneck and enabling early detection.

Our team consolidated and cleaned a dataset of **37,649 images** to train a robust model.

Example of a high-resolution fundus image used for training.

The Solution

A Clinical Decision Support System (CDSS)

Our team engineered an end-to-end Machine Learning model that classifies retinal scans, identifying 8 distinct pathologies with high accuracy.

1. Data Preprocessing

We cleaned, normalized, and augmented a dataset of **37,649** fundus images, handling significant class imbalances to prepare a robust dataset for training.

2. Model Training

We trained and fine-tuned a **DenseNet-121** Convolutional Neural Network (CNN) using **Keras/TensorFlow**, leveraging transfer learning for state-of-the-art feature extraction.

3. Classification

Our final model classifies images with **94.69% accuracy**, providing a reliable, automated "second opinion" to support medical professionals.

Future Work

Our Vision for What's Next

This project is a strong foundation. Our team has identified key areas for future improvement to create a truly clinical-grade system.

Implement Explainability (XAI)

Integrate Grad-CAM visualizations to create "heatmaps" showing *which* pixels the model used to make its decision, increasing clinical trust and transparency.

Add Severity Grading

Train new models to not only detect disease but to classify its severity (e.g., Mild, Moderate, or Severe Diabetic Retinopathy), providing more actionable insights.

Multi-Modal Architecture

Incorporate patient metadata (Age, Blood Pressure, HbA1c) alongside the image to create a holistic model that uses all available data for a more accurate diagnosis.
